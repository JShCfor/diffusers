## 可选环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `MODEL_ID` | `stable-diffusion-v1-5/stable-diffusion-v1-5` | 模型 ID 或本地路径 |
| `PORT` | `7860` | 端口号 |
| `SHARE` | `false` | `true` 时生成公网链接 |
| `HF_ENDPOINT` | — | 国内镜像加速 |

示例：

```bash
# 换端口
PORT=8080 python t2i_service.py

# 国内下载加速
HF_ENDPOINT=https://hf-mirror.com python t2i_service.py

# 使用本地已下载的模型
MODEL_ID=./models/sd-v1-5 python t2i_service.py
```
